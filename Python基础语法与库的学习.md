## 一、基础语法规则

### 1. 注释

| 单行注释     | `# 注释内容`                         | `# 这是单行注释` |

### 2. 缩进

- Python 用**缩进**区分代码块（而非大括号）
-  标准缩进：4 个空格（Typora 中按 Tab 键自动转换） 

## 二、变量与数据类型

### 1. 变量定义

- 规则：字母 / 下划线开头，区分大小写，不能用关键字

### 2. 基本数据类型

| 类型            | 说明                     | 示例                                                  |
| --------------- | ------------------------ | ----------------------------------------------------- |
| int（整数）     | 正 / 负整数、0           | `num = 10` `num = -5`                                 |
| float（浮点数） | 小数 / 科学计数法        | `pi = 3.14` `num = 1.2e3`                             |
| str（字符串）   | 单 / 双 / 三引号包裹     | `s1 = 'hello'` `s2 = "world"` `s3 = '''多行字符串'''` |
| bool（布尔）    | True/False（首字母大写） | `flag = True` `flag = 1 > 2  # False`                 |
| NoneType        | 空值（None）             | `res = None`                                          |

### 3. 类型转换

- 转整数 

  - int("123")  # 123 

  - int(3.9)    # 3（向下取整） 

- 转浮点数 

  - float("3.14")  # 3.14 

  - float(10)      # 10.0 

- 转字符串 

  - str(123)       "123" 
  - str(True)      # "True" 

  - bool(0)        # False 
  - bool("")       # False 
  - bool([])       # False 
  - bool(1)        # True

## 三、运算符

### 1. 算术运算符

| 运算符 | 说明               | 示例       | 结果 |
| ------ | ------------------ | ---------- | ---- |
| `+`    | 加                 | `10 + 20`  | 30   |
| `-`    | 减                 | `20 - 10`  | 10   |
| `*`    | 乘                 | `10 * 20`  | 200  |
| `/`    | 除（结果为浮点数） | `20 / 10`  | 2.0  |
| `//`   | 整除（向下取整）   | `21 // 10` | 2    |
| `%`    | 取模（余数）       | `21 % 10`  | 1    |
| `**`   | 幂                 | `2 ** 3`   | 8    |

### 2. 赋值运算符

- 若a = 10

- a += 5     等价于 a = a + 5 → 15
- a -= 3     等价于 a = a - 3 → 12
- a *= 2     等价于 a = a * 2 → 24
- a /= 4     等价于 a = a / 4 → 6.0
- a %= 2     等价于 a = a % 2 → 1.0

### 3. 比较运算符

| 运算符 | 说明     | 示例       | 结果 |
| ------ | -------- | ---------- | ---- |
| `==`   | 等于     | `10 == 10` | True |
| `!=`   | 不等于   | `10 != 20` | True |
| `>`    | 大于     | `20 > 10`  | True |
| `<`    | 小于     | `10 < 20`  | True |
| `>=`   | 大于等于 | `20 >= 20` | True |
| `<=`   | 小于等于 | `10 <= 20` | True |

### 4. 逻辑运算符

| 运算符 | 说明                  | 示例             | 结果  |
| ------ | --------------------- | ---------------- | ----- |
| `and`  | 与（全 True 才 True） | `True and False` | False |
| `or`   | 或（有 True 就 True） | `True or False`  | True  |
| `not`  | 非（取反）            | `not True`       | False |

# 库的学习

## Numpy

- NumPy(Numerical Python) 是 Python 语言的一个扩展程序库，支持大量的维度数组与矩阵运算，此外也针对数组运算提供大量的数学函数库。而如今很多深度学习框架也是基于NumPy的基础上进行开发的，因此支持从numpy数据对象(array)直接转换成框架特有的数据对象(tensor)
- 在实际应用中，常用NumPy来进行矩阵运算与相应的数据预处理

## Pandas

- Pandas是Python语言的一个扩展程序库， 一个强大的分析结构化数据的工具集，基于 Numpy开发。
- Pandas 可以从各种文件格式比如 CSV、JSON、SQL、Excel 导入数据。Pandas 可以对各种数据进行运算操作，比如归并、再成形、选择，还有数据清洗和数据加工特征。
- Pandas 广泛应用在学术、金融、统计学等各个数据分析领域。

## matplotlib

- matplotlib是Python语言的一个扩展程序库，是 Python 常用的 2D 绘图库，同时它也提供了一部分 3D 绘图接口。Matplotlib 通常与 NumPy、Pandas 一起使用，是数据分析中不可或缺的重要工具之一。

## JSON

- #### 简介：

  - JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式，它是JavaScript的子集，易于人阅读和编写。
  - JSON用来存储和交换文本信息，比xml更小/更快/更易解析，易于读写，占用带宽小，网络传输速度快的特性，适用于数据量大，不要求保留原有类型的情况。。
  - 前端和后端进行数据交互，其实就是JS和Python进行数据交互！

- #### 语法规则

  - 名称必须用双引号（即：" "）来包括
  - 值可以是双引号包括的字符串、数字、true、false、null、JavaScript数组，或子对象
  - 数据在name/value中
  - 数据见用逗号分隔
  - 花括号保存对象
  - 方括号保存数组

- #### 使用方法

  - son.dumps 将 Python 对象编码成 JSON 字符串
  - json.loads 将已编码的 JSON 字符串解码为 Python 对象
  - json.dump和json.load，需要传入文件描述符，加上文件操作。
  - json内部的格式要注意，一个好的格式能够方便读取，可以用indent格式化。
