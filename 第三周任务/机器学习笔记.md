# 机器学习笔记

## 一、机器学习基本概念

### 1.什么是机器学习

- 即让机器自动学习，而非人为规则编程
- 针对某任务 T，用经验 E 训练，性能 P 随 E 提升，即称程序从 E 中学习

### 2.基本概念学习

- 训练集：用于**训练**模型的数据集
- 测试集：用于**测试**模型的数据集
- 验证集：用于**验证**模型的数据集

- 有监督学习：输入数据是由输入特征值与目标值所组成，即输入的训练数据是有标签的
- 无监督学习：样本数据类别未知，没有标签，根据样本间的相似性，对样本进行聚类，以发现事物内部间的结构及相互关系

### 线性回归与逻辑回归特点与异同总结

#### 1. 相同点

- 都基于**线性组合**![image-20260405001546726](C:\Users\30346\AppData\Roaming\Typora\typora-user-images\image-20260405001546726.png)
- 都属于**广义线性模型**
- 都可用**梯度下降**训练
- 都易解释权重含义

#### 2. 不同点

| 对比项     | 线性回归           | 逻辑回归               |
| ---------- | ------------------ | ---------------------- |
| 任务类型   | 回归（连续值预测） | 分类（二分类为主）     |
| 输出范围   | R 任意实数         | (0,1) 概率             |
| 激活函数   | 无（恒等）         | Sigmoid                |
| 损失函数   | MSE（均方误差）    | 交叉熵（对数似然）     |
| 求解方式   | 闭式解 / 梯度下降  | 仅梯度下降（无闭式解） |
| 假设分布   | 高斯分布           | 伯努利分布             |
| 异常值敏感 | 高                 | 低（Sigmoid 压缩值域） |

### 3. K-Means 完整流程

1. 随机初始化 K 个中心
2. E 步：分配样本到最近中心
3. M 步：更新每个簇中心为均值
4. 重复 2–3 直到中心不再变化

------

### 4. 优劣势

### 优点

- 简单、快、易实现
- 可处理大规模数据
- 收敛快（通常几步稳定）

### 缺点

- 对**初始中心敏感**，易局部最优
- 对**噪声 / 异常值极敏感**
- 需**预先指定 K**
- 只能发现**凸簇、球形簇**，非凸 / 密度不均效果差
- 对**尺度敏感**，需标准化

## 四、PCA 降维

### 1. PCA 核心思想

将高维数据投影到**低维正交子空间**，保留**最大方差**，去除冗余。

### 2. PCA 缓解维度灾难的作用

- **降维**：减少特征数，降低存储 / 计算开销
- **去相关**：消除特征冗余与共线性
- **降噪**：小方差方向多为噪声，丢弃可提升泛化
- **加速模型**：减少过拟合，训练更快

### 3. PCA 在数据预处理中的作用

- **特征压缩**：高维→低维，保留主要信息
- **可视化**：高维→2D/3D 可画图
- **预处理**：常放在分类 / 聚类前，提升效果与稳定性
- **去噪**：保留主成分，抑制噪声
